CRG Collaborative Response Graphics

Description

  • Dustin Guillemin

  • www.crgplans.com

  • 2017-Present

At Critical Response Group Inc, Collaborative Response Graphics present a complex design challenge. The need for clear, rapid, and accurate communication by emergency responders means visual elements must be simple but effective. These products also need to be reliable in real-time, high-intensity situations. To solve these problems, the Collaborative Response Graphic is designed to be geospatially accurate, then further engineered with clearly recognizable icons and labels to facilitate user communication when it matters most.
